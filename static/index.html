StockWeb

Stock Price Query Service

Philippine Stock Exchange

About

StockWeb is an easy-to-use service that allows you to query stock prices for companies listed in the Philippine Stock Exchange.

Usage

http://stockweb-triapps.rhcloud.com/stock/{code}

Get back a JSON-formatted stock code and price
